# Digital-Voting
A secure full-stack digital voting web application that allows users to register, login, and cast votes online with authentication and database storage.
Digital Voting Platform is a full-stack web application designed to conduct elections online in a secure and transparent way.
The system allows users to register, login with authentication, view available candidates, and cast their vote digitally. Each user can vote only once, and all votes are stored safely in the database.

Features:
• User registration and login authentication
• Secure one-person-one-vote system
• Candidate listing and voting interface
• Vote storage in database
• Admin-controlled election management

Tech Stack:
Frontend: HTML, CSS, JavaScript
Backend: Xampp,Php
Database: MySQL

This project was developed as part of a Computer Science full-stack learning project.
